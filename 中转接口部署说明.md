# 中转接口部署说明

目标：让用户设备不直接访问 Google，而是提交到 Cloudflare Worker，再由 Worker 写入 Google 表格。

## 部署步骤

1. 打开 Cloudflare Workers 控制台，新建一个 Worker。
2. 把 `cloudflare-worker.js` 的全部代码复制进去。
3. 点击 Deploy。
4. 复制 Worker 的公开访问地址，通常类似：

```text
https://dxx-ordering-system.your-name.workers.dev
```

5. 把这个地址发给 Codex。

Codex 会把它写入 `index.html`：

```js
const ORDER_RELAY_ENDPOINT = "你的 Worker 地址";
```

之后网页提交路径就是：

```text
用户设备 -> Cloudflare Worker -> Google Apps Script -> Google 表格
```

这样用户设备不需要直接访问 `script.google.com`。
